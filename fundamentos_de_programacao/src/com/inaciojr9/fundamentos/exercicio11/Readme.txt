11) Substituição de letras

Faça um programa java que receba uma palavra digitada pelo usuário que contenha ao
menos uma letra a.
Caso a palavra digitada pelo usuário não contenha letra a, uma mensagem de erro deve ser
apresentada e o programa deve novamente solicitar ao usuário que digite uma palavra que
contenha a letra a. Após digitada uma palavra correta, o programa deve substituir todas as
letras "a" pela letra "o" e imprima na tela a palavra resultante.
